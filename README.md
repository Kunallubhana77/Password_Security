# Case Study 9: Password Manager

This project is a CLI-based Password Manager developed for **ITM Skills University**. It allows users to securely save, retrieve, and delete account credentials with basic encryption.

## Features
- **Save Password**: Store site name, username, and an encrypted password.
- **Retrieve Password**: Search for a site and view the decrypted password.
- **Delete Record**: Remove specific site records from the vault.
- **Security**: Uses a Shift Cipher (Shift +3) to store passwords securely in `vault.txt`.

## Files Used
- `password.cpp`: Main source code.
- `vault.txt`: Storage file for encrypted credentials.
